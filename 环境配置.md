### 1. 创建python虚拟环境
```psl
# 1. 进入项目目录
cd project-folder

# 2. 创建虚拟环境（名字叫 ai_env）
python -m venv ai_env

# 3. 激活虚拟环境
# Windows:
ai_env\Scripts\activate
# macOS/Linux:
# source venv/bin/activate

# 4. 安装依赖
pip install flask neo4j python-dotenv
pip install requests
pip uninstall flask python-dotenv
pip install fastapi uvicorn[standard] neo4j requests
```

### 2. 安装Neo4j数据库
* 打开网址`https://neo4j.com/download-center/#desktop`
* 找到 “Neo4j Desktop” 区块
* 点击对应系统的版本（Windows / macOS）
* 下载完成后，会得到一个安装文件
* 双击运行
* 打开desktop，create instance
* 设置数据库账号密码并记住（这个之后代码会用到）

### 3. 运行
* 首先需要运行main.py，执行：
  ```psl
  uvicorn main:app --reload --port 5000
  ```
* 然后打开
